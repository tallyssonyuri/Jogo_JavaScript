O DESAFIO

"O desafio" é um game no qual o jogador precisa desviar por uma série de obstáculos para chegar em um objetivo final. Ao encostar em algum deles, o jogador voltará para a posição inicial, precisando refazer o mesmo caminho

Controles

Para jogar é bem simples, basta utilizar o botão space e o jogo vai iniciar.

Para direcionar o personagem é possível com duas formas: utilizando teclas direcionais do teclado ou WASD (w = cima, a = esquerda, s = baixo, d = direita).

A tecla esc pausa o game e a tecla R reinicia.
